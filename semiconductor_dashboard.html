Project Overview
Objective

Build an auditable, SEC-sourced equity-research workflow for five public semiconductor companies. Every financial metric traces back to an SEC EDGAR filing URL. The pipeline distinguishes reported values from derived values, flags missing data for human review, and separates sourced facts from AI-generated interpretation.

Qualcomm served as the initial regression benchmark (132 financials). After each company addition, automated checks confirmed identical Qualcomm output before proceeding.

Methodology

Data extraction: XBRL quarterly extraction from 10-K and 10-Q filings via SEC EDGAR. Q4 values derived as full-year minus 9-month YTD where standalone Q4 data is unavailable.

Metrics: 15 financial metrics per company per quarter, including revenue, margins, FCF, R&D, and balance-sheet items. TTM aggregations computed from trailing four quarters.

Valuation: Share prices from Yahoo Finance (June 4, 2026). Shares outstanding from most recent SEC filing cover pages. Enterprise value = market cap + total debt − cash.

Qualitative analysis: 40 source-linked claims extracted from 10-K filings, each labeled factual or interpretive with confidence ratings and human-review status.
